# 🌐 kolyamips Portfolio

Personalized high-performance portfolio website built with React, Vite, and Tailwind CSS. Featuring 3D background shapes, interactive rain overlays, and an integrated audio visualizer.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🛠 Tech Stack

- **Framework:** React 19
- **Bundler:** Vite 7
- **Styling:** Tailwind CSS 4
- **Animations:** CSS 3D Transforms
- **Deployment:** Vercel
